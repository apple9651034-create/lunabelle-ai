# 육효 리딩 개선 계획

## 현재 문제점

1. **괘 정보 부족**
   - 괘의 이름 없음 (예: 건괘, 곤괘 등)
   - 괘의 상징 설명 없음
   - 괘의 의미 미흡

2. **변효 해석 부족**
   - 변효의 의미만 반복
   - 미래 변화 구체성 없음
   - 행동 방향 제시 미흡

3. **질문별 맞춤 해석 부족**
   - 일반적 조언만 제시
   - 사업운, 연애운 등 구체적 해석 없음
   - 질문 맥락 무시

4. **구체적 실천 방법 부족**
   - "적극적 행동하세요" 같은 추상적 조언
   - 구체적 행동 방안 없음
   - 주의사항 미흡

## 개선 방안

### 1. 괘 정보 추가

```
본괘: 건괘(乾卦)
- 상징: 하늘, 창조, 강함, 리더십
- 의미: 순수한 양의 에너지, 창조의 시작, 강한 의지
- 해석: [질문에 맞는 구체적 해석]

변괘: 곤괘(坤卦)
- 상징: 땅, 수용, 유연함, 포용
- 의미: 순수한 음의 에너지, 포용과 수용, 안정성
- 해석: [미래 변화에 맞는 구체적 해석]
```

### 2. 변효 해석 고도화

```
제6효 - 양효 (변효)
- 위치: 최상단 (결과/미래)
- 의미: 상황의 극단, 변화의 정점
- 해석: [구체적 미래 변화]
- 주의: [주의사항]
- 행동: [구체적 행동 방안]
```

### 3. 질문별 맞춤 해석

**사업운 질문 예시:**
- 현재 사업 상황 분석
- 사업 확장/축소 판단
- 파트너십 관계
- 자금 흐름
- 구체적 조언

**연애운 질문 예시:**
- 현재 관계 상황
- 미래 발전 방향
- 주의사항
- 구체적 행동

### 4. 구체적 실천 방법

```
【현재 해야 할 일】
1. [구체적 행동 1]
2. [구체적 행동 2]
3. [구체적 행동 3]

【피해야 할 일】
1. [피해야 할 행동 1]
2. [피해야 할 행동 2]

【마음가짐】
- [긍정적 마음가짐]
- [주의할 점]
```

## 구현 방법

1. 육효 결과 구조 변경 (YukConsultationPage.tsx)
2. 괘 정보 데이터베이스 확장 (yukPrompt.ts)
3. 질문 분석 시스템 추가
4. 맞춤 해석 템플릿 구축
5. 실천 방법 가이드 추가
